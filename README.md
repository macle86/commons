# commons
- runon 개발에 사용되는 공통모듈 모음 프로젝트
- 1.8 버전으로 개발을 우선시 합니다.
  - 최신버전으로 하려고 했지만, 기존 다른프로젝트들이 1.8 버전이 많아서 공통 버전은 가능하면 1.8 버전대로 합니다. 
- 2개 이상의 프로젝트에서 사용되는 모듈들 입니다.
- 공통모듈은 modules에 존재 합니다.
- 각 공통 모듈의 사용법은 모듈의 README.md 파일을 참고하세요.

## 개발환경
- open jdk 1.8
